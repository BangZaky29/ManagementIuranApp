# Warlok (Warga Lokal) 👋

Aplikasi Manajemen Iuran dan Komunikasi Warga yang efisien dan modern.

## Fitur Utama
- **Iuran**: Pembayaran iuran keamanan & sampah secara digital.
- **Laporan**: Pengaduan rintangan atau masalah lingkungan.
- **Buku Tamu**: Registrasi tamu dengan kode PIN.
- **SOS**: Tombol darurat untuk keamanan komplek.

## Get started

1. Install dependencies
   ```bash
   npm install
   ```

2. Start the app
   ```bash
   npx expo start
   ```
